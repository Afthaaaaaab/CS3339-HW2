# CS3339 Homework 2 - Floating Point Overflow Checker

## Team Members
- Munavvir Afthab Shabab

## Description
This program detects potential floating-point overflow by analyzing
the IEEE-754 32-bit representation of two floating-point values.

## Compilation
Use g++ to compile the program:

g++ implementation_file.cpp -o implementation_file

## Running the Program

Run the executable with two floating point numbers:

./implementation_file <loop_bound> <loop_counter>
