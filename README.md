# CS3339 Homework 2 - Floating Point Overflow Checker

## Team Members

## Description
This program detects potential floating-point overflow by analyzing
the IEEE-754 32-bit representation of two floating-point values.

## Compilation
Use g++ to compile the program:

g++ fp_overflow_checker.cpp -o fp_overflow_checker
